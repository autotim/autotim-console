# Historical Documents

These two documents record the critical review that produced **Architecture
Foundation v2** (the live design in `docs/architecture/`). They are kept for
context — *why* the v2 decisions were made — not as current reference.

| File | What it is |
|------|-----------|
| `ARCHITECTURE_REVIEW.md` | Critical, principal-level review of Foundation **v1** (18 findings: tenancy, module model, async substrate, RBAC, secrets, etc.) |
| `ARCHITECTURE_IMPROVEMENT_PLAN.md` | The five foundational decisions and commit plan that resolved those findings into v2 |

> **Numbering note:** these documents reference the **original v1 numbering**
> (`doc 00`–`doc 20`, the pre-rewrite Foundation), which no longer exists as
> files in this repository. Do not look for `docs/architecture/06-*.md` based
> on a reference here — the current, canonical document set and its banded
> numbering (`00, 10–14, 20–24, 30–32, 40–42, 50–51, 60–62`) lives in
> `docs/architecture/`, starting at `INDEX.md`.
